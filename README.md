# Michelle Fernandez Portfolio

This is the personal portfolio website of **Michelle Fernandez**, a UI/UX Designer and Front-End Developer. The website showcases Michelle's skills, projects, education, and contact information in a clean and responsive design.

## Features

- **Hero Section**: Introduction with a profile image and call-to-action buttons.
- **About Section**: Personal information and core skills.
- **Skills Section**: Technology stack and tools with interactive hover effects.
- **Projects Section**: Highlights three main projects with descriptions and demo links.
- **Education Section**: Academic background with icons and timeline.
- **Contact Section**: Contact details including phone, email, GitHub, and LinkedIn.
- **Responsive Design**: Works on desktop, tablet, and mobile devices.
- **Smooth Animations**: Hover effects and transitions for interactive UI.

## Tech Stack

- **HTML5** – Structure of the website
- **CSS3 / Tailwind CSS** – Styling and responsive design
- **JavaScript** – Mobile menu toggle
- **Font Awesome** – Icons for skills, projects, and contact info
- **GitHub Pages** – Hosting and deployment
